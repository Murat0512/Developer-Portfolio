SiteTracker — Full Stack Site Progress Tracker

Angular 17 • Node.js/Express • MongoDB • PDFKit • Cloudinary • Vercel

Live Demo

Problem

Site progress and evidence often get scattered across spreadsheets, photos, and messages.

Solution

  • Daily log creation with filters and date-range querying
  • Evidence attachments (images/PDFs) via Cloudinary
  • PDF reporting by date range (PDFKit)
  • JWT authentication and protected routes
  • Deployed frontend workflow on Vercel

CivTech Auditor Pro — As-Built Verification Tool

Python • Streamlit • Pandas • Plotly • ezdxf • XlsxWriter

Live Demo (Public)

Problem

Survey QA against CAD design is time-consuming and difficult to audit consistently.

Solution

  • 3D deviation checks: horizontal (plan) + vertical (elevation)
  • Coordinate nudge system to align survey to design datum
  • Interactive overlay view with pass/fail markers
  • Audit exports concept: verified report + color-coded DXF output
